🔄 Agent calls /reset POST

The AI agent starts a new episode by calling the /reset endpoint. The environment generates a fresh 100-row employee dataset with injected data quality issues and returns Task 1.

🧠 LLM analyzes the dataset LLM

The agent feeds the CSV dataset and task description to the LLM using the OpenAI client. The model reasons about the data and produces a structured JSON response.

inference.py — LLM CallPython
response = client.chat.completions.create(
  model=MODEL_NAME,
  messages=[
    {"role": "system", "content": "You are an expert data analyst..."},
    {"role": "user", "content": task_prompt + dataset_csv}
  ],
  response_format={"type": "json_object"},
  temperature=0.1
)

⚡ Agent submits to /step POST

The agent posts its action JSON to /step. The environment grader scores the response and returns a reward between 0.0 and 1.0 plus the next task observation.

REQUEST — /stepJSON
POST /step
{ "action": { ...agent's JSON response... } }
RESPONSE — StepResultJSON
{
  "reward": 0.920,
  "done": false,
  "info": {
    "task_id": "task_easy",
    "feedback": "✓ All missing value columns identified | ✓ 6/6 types found"
  },
  "observation": { // next task... }
}
